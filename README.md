# PeakUSB
A bootable USB creation tool. Fully open-source.

## -Overview-
PeakUSB contains all the basic features of a USB creation tool, but adds in plenty of advanced
features, such as:

-Checksum verification after ISO writing

-Bad block checking

-Persistent Partition Creation
and more.

PeakUSB is currently in its early stages, and may not fully function. Report any bugs.

If you would like to become a contributor, feel free to put that in the bugs page (not giving you my personal email ;])
